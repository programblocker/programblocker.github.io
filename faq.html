Find answers to the most common questions about ProgramBlocker — installation, blocking, licensing, troubleshooting, and more.

▼ General
Q: What is ProgramBlocker?
A: ProgramBlocker is a powerful Windows utility that lets you block any .exe program from accessing the internet or from running entirely. It is designed for productivity, parental control, and system security — no technical knowledge required.
Q: What are the system requirements?
A: ProgramBlocker works on:
  • Windows 7 / 8 / 8.1 / 10 / 11 (both 32-bit and 64-bit)
  • Administrator privileges (required to apply firewall rules)
  • No additional runtimes or frameworks needed
Q: Does blocking a program affect other apps or my internet connection?
A: No. Only the specific .exe files you choose to block are affected. All other programs and your overall internet connection remain completely unaffected.
Q: Is ProgramBlocker safe? Does it contain ads or malware?
A: Yes, ProgramBlocker is completely safe. It contains no ads, no malware, no spyware, and no bundled software. It only modifies Windows Firewall rules and does not collect any personal data.
▼ How to Block Programs
Q: How do I add a program to the blocklist?
A: Click the "Select .EXE (Program)" button (or press CTRL + O) in the main interface, then browse to and select one or more .exe files. After selecting, choose:
  • Block Internet — prevents the program from accessing the internet
  • Block Execution — prevents the program from launching at all
  • Or select both for maximum control
Q: Can I block multiple programs at once?
A: Yes. When using the file picker (CTRL + O), you can hold Ctrl or Shift to select multiple .exe files simultaneously. All selected files will be added to the blocklist in one step.
Q: What is the difference between "Block Internet" and "Block Execution"?
A:
  • Block Internet — The program can still run and open, but it cannot send or receive any data over the internet. This is useful for apps you want to use offline, or to stop unwanted updates and telemetry.
  • Block Execution — The program cannot launch at all. Any attempt to open it will be silently prevented. This is useful for parental control or preventing specific software from ever running.
Q: How do I remove a block from a program?
A: Select the program from the blocklist, then click the appropriate button:
  • Allow Internet — restores internet access for that program
  • Allow Execution — allows the program to run again
  • Unblock — removes all blocks (internet + execution) from the selected program at once
Q: Will blocks survive a Windows restart or update?
A: Yes. Blocks applied by ProgramBlocker use Windows Firewall rules, which persist across reboots and Windows updates. You do not need to re-apply blocks after restarting your PC.
Q: Can I block a program that is currently running?
A: Yes. You can add a running program to the blocklist. The internet block takes effect immediately. For execution blocking, the program must be restarted — the block will apply the next time it tries to launch.

▼ Troubleshooting
Q: The block doesn't seem to be working. What should I check?
A: If a block does not appear to work, try the following:
  • Make sure you launched ProgramBlocker as Administrator (right-click → Run as Administrator)
  • Confirm that Windows Firewall is enabled on your system — ProgramBlocker uses it to enforce internet blocks
  • Verify that you blocked the correct .exe file — some programs use multiple executables
  • Restart the target program after applying the block
  • If issues persist, contact programblocker@gmail.com
Q: A blocked program is still accessing the internet. Why?
A: This can happen if the program uses a different .exe or a helper process to connect. Some applications launch a separate updater or service executable. Try:
  • Identifying all related .exe files for that application (check Task Manager while the program is running)
  • Blocking each related executable individually
Q: ProgramBlocker won't open or crashes on startup. What should I do?
A: Try the following steps:
  • Right-click ProgramBlocker and select "Run as Administrator"
  • Make sure you are running a supported version of Windows (7, 8, 10, or 11)
  • Try re-downloading the latest version from the official website
  • Temporarily disable antivirus to check if it is blocking the app
  • If the issue continues, email us at programblocker@gmail.com with details about your Windows version and the error message (if any)
